🛠️ STACK TECHNIQUE
═══════════════════════════════════════════════════════════════════════════

Framework UI:
  • Tkinter (interface)
  • Pillow (affichage image)

Vision:
  • OpenCV 4.5+ (caméra + détection)
  • NumPy (transformations images)
  • Haar Cascade (pré-entraîné)

Audio:
  • sounddevice (capture)
  • soundfile (WAV I/O)
  • SciPy (traitement signal)

Communication:
  • requests (HTTP client)
  • Base64 (encodage)

Utilitaires:
  • threading (parallelisme)
  • logging (traces)
  • python-dotenv (config)


📊 PERFORMANCE
═══════════════════════════════════════════════════════════════════════════

Caméra:
  Resolution: 640x480
  FPS: ~30
  Latency: ~33ms/frame
  CPU: 10-15%

Audio:
  Sample rate: 44.1kHz
  Duration: 5 seconds
  Size: ~440KB WAV / ~590KB Base64
  Processing: ~100ms

API:
  Network latency: 50-200ms
  Server processing: 2-5 secondes
  Total auth: 5-15 secondes

Mémoire:
  Base: ~50MB
  + Caméra: +70MB
  + Audio: +50MB
  Peak: ~200MB
