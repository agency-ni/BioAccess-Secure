# 🚀 INSTALLATION - BioAccess Secure

## ⚠️ IMPORTANT - LISEZ CECI EN PREMIER!

Vous n'avez **besoin que d'UN SEUL FICHIER** pour tout installer.

---

## 📋 Selon votre système d'exploitation:

### 🪟 WINDOWS

1. **Double-cliquez** sur le fichier: `SETUP_WINDOWS.bat`
2. Suivez les instructions à l'écran
3. C'est tout! ✅

#### ⚠️ Si ça ne marche pas:
```
→ Vérifiez que Python est installé:
  https://www.python.org/downloads/
  (Cochez "Add Python to PATH" lors de l'installation)
```

---

### 🐧 LINUX

1. **Ouvrez un terminal** dans ce dossier
2. **Donnez les permissions** au script:
   ```bash
   chmod +x SETUP_LINUX_MAC.sh
   ```
3. **Lancez l'installation:**
   ```bash
   ./SETUP_LINUX_MAC.sh
   ```
4. Suivez les instructions
5. C'est tout! ✅

---

### 🍎 macOS

1. **Ouvrez un terminal** dans ce dossier
2. **Donnez les permissions** au script:
   ```bash
   chmod +x SETUP_LINUX_MAC.sh
   ```
3. **Lancez l'installation:**
   ```bash
   ./SETUP_LINUX_MAC.sh
   ```
4. Suivez les instructions
5. C'est tout! ✅

---

## ✅ Résumé - 3 étapes simples

```
Windows:      Double-cliquez sur SETUP_WINDOWS.bat
Linux/macOS:  ./SETUP_LINUX_MAC.sh
```

**C'est automatique!** L'installation détecte votre système et configure tout.

---

## 📞 Ça ne marche pas?

**Avant de continuer, s'assurer que:**

✅ Python 3.7+ est installé
✅ Vous avez une connexion Internet (pour télécharger les modules)

**Erreur "Python not found"?**
→ https://www.python.org/downloads/ (Windows: cochez "Add to PATH")

**Erreur "Permission denied"?**
→ Exécuter: `chmod +x SETUP_LINUX_MAC.sh`

## 🎯 Après l'installation
