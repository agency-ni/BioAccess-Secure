📌 FICHIERS CRÉÉS / MODIFIÉS

1️⃣  START.bat (POINT D'ENTRÉE UNIQUE)
────────────────────────────────────────────────────────────────
   📋 Description: Menu interactif pour tous les utilisateurs
   👥 Public: TOUT LE MONDE (utilisateurs, devs, admins)
   ⚡ Action: Double-cliquer ICI en premier!
   
   Propose 4 options:
     • Installation utilisateur final (setup_wizard.bat)
     • Installation développeur (install.bat)
     • Compilation .exe (build.bat)
     • Lire documentation (INSTALLATION.txt)


2️⃣  setup_wizard.bat (INSTALLATION COMPLÈTE - UTILISATEURS)
────────────────────────────────────────────────────────────────
   📋 Description: Installation totale de A à Z
   👥 Public: Utilisateurs finaux (0 connaissance IT requise)
   ⚡ Étapes:
     1. Vérifie Python (installe si besoin)
     2. Crée environnement virtuel
     3. Installe toutes les dépendances
     4. Compile l'app en .exe with PyInstaller
     5. Demande où installer
     6. Crée raccourci desktop
     7. Prêt à utiliser!
   
   ⏱️ Durée: ~5-10 minutes
   🎯 Résultat: Application prête dans C:\Program Files\ et raccourci Bureau


3️⃣  build.bat (COMPILATION .EXE)
────────────────────────────────────────────────────────────────
   📋 Description: Crée BioAccessSecure.exe autonome
   👥 Public: Développeurs, administrateurs
   ⚡ Nécessite: install.bat lancé d'abord
   
   Procédure:
     • Nettoie les anciens builds
     • Lance PyInstaller avec config optimisée
     • Génère dist/BioAccessSecure.exe
   
   📦 Résultat: Dossier dist/ avec .exe autonome
   📤 Distribution: Copier dist/ n'importe où pour partager


4️⃣  install.bat (INSTALLATION DÉVELOPPEUR)
────────────────────────────────────────────────────────────────
   📋 Description: Setup classique pour développement
   👥 Public: Développeurs, contributeurs
   ⚡ Résultat: Environnement venv + dépendances
   
   Après:
     • Modifier .env
     • Lancer: python main.py


5️⃣  main.spec (CONFIGURATION PYINSTALLER)
────────────────────────────────────────────────────────────────
   📋 Description: Fichier config pour compilation
   👥 Utilisé par: build.bat automatiquement
   🔧 Configure:
     • Fichiers inclus (ui/, biometric/, services/)
     • Modules cachés (cv2, scipy, sounddevice, etc.)
     • Options compilation (onefile, windowed, etc.)


6️⃣  requirements-optional.txt (MODIFIÉ)
────────────────────────────────────────────────────────────────
   📋 Description: Dépendances pour compilation + ML
   ➕ Ajouté: PyInstaller (pour build.bat)
   
   Contient:
     • pyinstaller>=5.0.0 (compilation)
     • scikit-learn>=1.0.0 (ML optionnel)


7️⃣  INSTALLATION.txt (GUIDE COMPLET)
────────────────────────────────────────────────────────────────
   📋 Description: Documentation détaillée
   👥 Public: Tous les utilisateurs
   📚 Sections:
     • 3 façons d'installer
     • Pré-requis
     • FAQ & Troubleshooting
     • Documentation disponible

🚀 DÉMARRAGE RECOMMANDÉ POUR CHAQUE PROFIL
════════════════════════════════════════════════════════════════

UTILISATEUR FINAL:
  1. Lancer START.bat
  2. Choisir option 1 (utilisateur final)
  3. Attendre fin installation
  4. Double-cliquer raccourci Bureau
  ✅ C'est prêt!

DÉVELOPPEUR:
  1. Lancer START.bat
  2. Choisir option 2 (développeur)
  3. Modifier .env (API_BASE_URL)
  4. Lancer: python main.py
  ✅ Mode développement!

COMPILEUR:
  1. Lancer START.bat
  2. Choisir option 2 (install dev)
  3. Lancer START.bat à nouveau
  4. Choisir option 3 (build)
  5. Attendre compilation
  ✅ dist/BioAccessSecure.exe prêt!


❓ QUESTIONS FRÉQUENTES
════════════════════════════════════════════════════════════════

Q: Par où commencer?
A: START.bat → Vous indiquera quoi faire

Q: Installation prend trop longtemps?
A: Normal! OpenCV + scipy = ~500MB. Patient 10-15 min.

Q: Erreur "Python not found"?
A: https://www.python.org → Cocher "Add to PATH"

Q: L'exe ne démarre pas?
A: Vérifier .env (API_BASE_URL doit être valide)

Q: Où sont mes fichiers config?
A: Dans le dossier d'installation + .env
